# Feature 1.14 — Creator Onboarding Flow

**Branch:** `feature/1.13-1.14-onboarding`
**Roadmap ref:** Phase 1 · Feature 1.14 — Creator onboarding flow
**Status:** 🟢 Complete

## Overview

4-step onboarding wizard for creators after registration. Users who register with the `creator` role are redirected to the onboarding flow before they can access the dashboard or any protected route.

## Implementation

### Controller

`App\Http\Controllers\Onboarding\CreatorOnboardingController` — handles the 4-step flow:

1. **Personal profile** (`GET/POST /onboarding/creator/profile`) — display name (required), bio (optional), country code (optional). Creates or updates `CreatorProfile` via `updateOrCreate`. Also updates user's `country` field.
2. **Niche selection** (`GET/POST /onboarding/creator/niches`) — multi-select niche picker (min 1, max 5). Uses `sync()` on the `creator_niches` pivot table. Niches loaded from `niches` table.
3. **Social accounts** (`GET/POST /onboarding/creator/social`) — placeholder step for social OAuth connections (Phase 5). Currently skippable with "Continue — connect accounts later" button.
4. **Payout setup** (`GET /onboarding/creator/payout`, `POST /onboarding/creator/complete`) — feature tour showing earning methods (Contests, Ripple, Pitch, Secure payments). Completing this step sets `onboarding_completed_at` and redirects to dashboard.

Each step checks if the user is already onboarded and redirects to dashboard. Steps 2–4 redirect to step 1 if no profile exists yet.

### Routes

All routes under `onboarding/creator/*` prefix, protected by `auth`, `verified`, and `role:creator` middleware. Defined in `routes/onboarding.php`.

### Frontend

- **Profile page** — form with display name (pre-filled from user name), bio textarea, country code input (auto-uppercase, 2 chars max).
- **Niches page** — pill-style toggle buttons for all active niches. Visual counter (X/5 selected). Disabled state when max reached.
- **Social page** — informational card explaining social account connections with skip option.
- **Payout page** — feature cards (Win contests, Earn per view with Ripple, Pitch to brands, Get paid securely) with "Go to dashboard" button.
