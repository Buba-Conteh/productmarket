# Feature 1.9 — Seeders for Lookup Tables

**Branch:** `feature/1.9-1.10-seeders-auth`
**Roadmap ref:** Phase 1 · Feature 1.9 — Seeders — lookup tables
**Status:** 🟢 Complete

## Overview

Seed data for all four lookup tables was implemented as part of feature 1.3. This feature confirms the seeders are wired, idempotent, and running cleanly.

## Seeders

| Seeder | Table | Records |
|---|---|---|
| `PlatformSeeder` | platforms | TikTok, Instagram, YouTube |
| `NicheSeeder` | niches | 12 niches (Fitness, Beauty, Tech, etc.) |
| `IndustrySeeder` | industries | 12 industries (Health & Wellness, Fashion, etc.) |
| `ContentTypeSeeder` | content_types | 6 types (GRWM, Unboxing, Demo, etc.) |
| `PlatformSettingSeeder` | platform_settings | Single-row config (added in 1.7) |

All seeders use `updateOrCreate` keyed on `slug` for idempotency — safe to re-run.

## Verification

- `migrate:fresh --seed` runs clean.
- All 40 Pest tests pass.
